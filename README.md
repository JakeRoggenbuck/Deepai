# Deepai

## Use

```
from main import Deep

key = "api-key"
url = "https://api.deepai.org/api/deepdream"
file = "image.png"
path = "images/"

deep = Deep(key, url, file, path)
deep.download()
```

## Config
Make the config and get an api key at [deepai.org](https://deepai.org)
`config.py`
```
KEY = "API-KEY"
URL = "https://api.deepai.org/api/deepdream"
```
